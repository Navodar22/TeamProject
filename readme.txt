1. Pred kazdou upravou projektu treba zo serveru stiahnut nove zmeny od ostatnych
	TortoiseGit -> Pull

2. Najprv každí pracuje na localhoste. Prikaz "Git commit -> branch" je lokalny. Takže zmeny sa na server neuploadnu

3. Upload na server prebieha prikazom 
	TortoiseGit -> Push
	Ak to nieje potrebné nevytvárajte nové branche.

4. Na server commitujte iba hotove veci a ziadne nedokonèené úpravy nech je v tom poriadok !!!
